# GosuslugiKids_bot
Телеграм-бот на pyTelegramBotAPI, позволяющий ребёнку регистрироваться в различных секциях дополнительного образования с помощью сертификата ПФДО.
Проект создан во время проведения хакатона KD-HACK2021
